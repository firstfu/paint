## Customization Guide

### Style Keywords

| Category | Options |
|----------|---------|
| Art Style | flat, 3D, isometric, hand-drawn, pixel art, watercolor, vector, gradient |
| Mood | professional, playful, elegant, modern, vintage, futuristic, minimal |
| Colors | vibrant, pastel, monochrome, gradient, neon, muted, warm, cool |

### Quality Modifiers

Add these to improve output quality:

- "high detail"
- "sharp edges"
- "clean lines"
- "professional quality"
- "production ready"
- "polished finish"

### Negative Prompts

Specify what to avoid:

- "no text"
- "no watermarks"
- "avoid clutter"
- "no busy backgrounds"
- "simple composition"

## Template Usage Tips

1. **Replace All Placeholders**: Don't leave brackets in final prompt
2. **Be Specific**: More detail usually yields better results
3. **Specify Aspect Ratio**: Match your actual requirements
4. **Include Brand Details**: Colors, style guidelines
5. **Iterate**: Generate variations and refine prompts
6. **Save Successful Prompts**: Build a library for consistency
